# Customer-specific Sentinel content

Each customer folder follows this naming convention:

- `customer-<SHORTNAME>`

Each customer folder includes these content subfolders:

- `analytic-rules/`
- `automation-rules/`
- `hunting-queries/`
- `parsers/`
- `playbooks/`
- `workbooks/`
